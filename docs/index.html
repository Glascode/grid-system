Grid control

Columns control
c-2
start at next column,
span 2 columns.
c-2-5
start at column 2,
end at column 5.
c-2-e
start column 2,
end at last column.
c-2--3
start at column 2,
span 3 columns.
Rows control
r-2
start at current row,
span 2 rows.
row
span a full row.
