Grid control

Columns control

span-2
start at next column,
span 2 columns.
span-2-5
start at column 2,
end at column 5.
span-2..
start column 2,
end at last column.
span-2+3
start at column 2,
span 3 columns.
span-row
span a full row.

Rows control

vspan-2
start at current row,
span 2 rows.

Responsiveness

{{ site.title }} uses a mobile first breakpoint system, similar to other popular CSS frameworks. Set a responsive behavior with a selector prefix followed by :, the defaults are sm, md and lg.
