Grid control

Columns control
c-2
start at next column,
span 2 columns.
c-2-5
start at column 2,
end at column 5.
c-2-e
start column 2,
end at last column.
c-2--3
start at column 2,
span 3 columns.
Rows control
r-2
start at current row,
span 2 rows.
row
span a full row.

Responsiveness

Matrix uses a mobile first breakpoint system, similar to other popular CSS frameworks. Set a responsive selector among l, m and s. 

<div class="grid columns-3 example">
  <div class="row-m">Grid</div>
  <div class="c-2 row-m">Home &nbsp; About &nbsp; Contact</div>
  <div class="row-m">Menu</div>
  <div class="c-2 row-m">Welcome</div>
</div>
