Grid control

Columns control

spanx-2
start at next column,
span 2 columns.
spanx-2-5
start at column 2,
end at column 5.
spanx-2..
start column 2,
end at last column.
spanx-2+3
start at column 2,
span 3 columns.
spanx-row
span a full row.

Rows control

vspanx-2
start at current row,
span 2 rows.

Responsiveness

Matrix uses a mobile first breakpoint system, similar to other popular CSS frameworks. Set a responsive behavior with a selector prefix followed by :, the defaults are sm, md and lg.
